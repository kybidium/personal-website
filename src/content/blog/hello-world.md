Welcome! I'm Luis, and this is my new home on the internet.

I've been meaning to carve out a spot to write for a while. This site is where I'll post about software I'm building, things I'm learning, and the occasional off-topic tangent.

## What to expect

- Notes on projects I'm working on
- Things I learned the hard way
- Whatever else seems worth writing down

To add a new post, I just drop a markdown file into `src/content/blog/` — no CMS, no database, nothing to break.
